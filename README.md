# 📊 Inventory Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive Inventory Dashboard developed using Power BI to analyze stock levels, demand forecasts, and product performance.  
The dashboard provides meaningful insights to support data-driven decision making.

## 🛠 Tools & Technologies Used
- Power BI  
- Data Visualization  
- Data Analysis  

## 📈 Key Features
- Inventory Level Tracking  
- Demand Forecast Analysis  
- Units Sold & Ordered Insights  
- Product Category Performance  
- Warehouse-wise Analysis  

## 🎯 Objectives
- Monitor inventory efficiently  
- Identify demand patterns  
- Improve business planning
